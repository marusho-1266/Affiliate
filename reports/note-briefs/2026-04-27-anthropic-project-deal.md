# 報告資料: Anthropic「Project Deal」— エージェント同士の社内マーケット実験

- **調査日**: 2026-04-27
- **主出典（一次）**: [Anthropic — Project Deal](https://www.anthropic.com/features/project-deal)（ページ掲載 `posted: April 24, 2026`）
- **補助（二次）**: [TechCrunch](https://techcrunch.com/2026/04/25/anthropic-created-a-test-marketplace-for-agent-on-agent-commerce/)（2026-04-25 公開。要約・呼称の裏取り用）
- **要約（1文）**: Anthropic がサンフランシスコ拠点の従業員69名を対象に、Slack 上で「人の代わりに AI エージェントだけが売買交渉する」分類広告型の1週間パイロット（Project Deal）を実施し、取引成立件数・金額、Opus と Haiku の交渉性能差、当事者の主観とのズレなどを公表した研究寄りの実験報告である。

## 3. 効果 — 短期・中期・長期の予想

**時間軸の定義（本資料）**  
- **短期**: 0〜3か月  
- **中期**: 3か月〜1年  
- **長期**: 1年以上  

※以下の「市場・製品」への影響は、**公表内容があくまで社内パイロット**であることを前提とした**シナリオ**であり、投資助言・売上予測ではない。

- **短期（0〜3か月）**  
  - **予想される効果**: 学術界・SNS・開発者層での議論、Anthropic 研究ブランドの露出。他社の**類似デモ**や批評コメントの増加。  
  - **不確実性**: メディアが数値（186 件、約 4,000 ドル等）を**要約誤り**しうる。  
  - **効果が出ない条件**: 大きな消費者向けプロダクト投入が伴わない場合、熱点は一瞬で減衰しうる。

- **中期（3か月〜1年）**  
  - **予想される効果**: 「エージェント格差（agent quality）」をキーワードに、価格・労働・規制（消費者保護、代理行為、広告表示）の議論に接続されうる。  
  - **不確実性**: 実験の母集団は**自社従業員・自選**であり、市場一般への外挿は官方も慎重（本文で「pilot」「self-selected」と明記）。  
  - **効果が出ない条件**: 管轄官庁のルール作りが遅れ、議論が学会・業界内に閉じる。

- **長期（1年以上）**  
  - **予想される効果**: 人間が直接交渉せず、モデル階層（商用 API のティア、オープン重み等）が**価格交渉力**に直結しうる、という仮説の常識化／批判の定着。  
  - **不確実性**: モデル更新で交渉能力の相対差が変わる。介入規制（AI が契約に署名する等）の有無。  
  - **効果が出ない条件**: エージェント同士の自律交渉が実務で普及しない、または人間承認の義務化が徹底され、実験で見た「無介入」状況が再現されない。

# note 三層パック（本編＋補助＋筆者メモ・有料想定）

**本編**（無料で論点・**時間軸の観察**まで完結）＋**補助**（無料で用語・読み順・レイヤー）＋**筆者メモ**（note 上で有料ブロック想定、**内部骨格は 7-1〜7-5**）の役割分担で `note/` 底稿を整理する。実行時は **`.cursor/skills/note-trio-pack/SKILL.md`** に従う（**観察 vs 判断**の振り分け、**7-1〜7-5** の雛形、品質チェックは SKILL 側が正本）。

## いつ使うか

- 「難しい」対策で**補助記事**を増やしたいとき
- 既存本編に**読み順**と**補助へのリンク**、**主な参照（出典）の直後**に**筆者メモ（有料想定）**の見出しを足したいとき
- 既存本編を**有料化レディ**に再編したいとき（効果の「順序づけ・比率・役割別打ち手」を筆者メモに移し、本編の時間軸は**観察レーン**だけを残す）
- 新規トピックで、最初から三層でファイル名を揃えたいとき

## 入力

- **本編**のパス（必須）。任意で**補助**のファイル名・**筆者メモ**に入れたい論点リスト（7-1〜7-5 のどこに入れたいかを添えると変換が正確になる）。

## 出力

- ユーザーの指示に従い、**補助 `.md` の新規作成**、**本編の追記**、**有料化レディへの再編**（無料の観察レーン整理＋ `## 〈番号〉. 筆者メモ（…）` ブロックの 7-1〜7-5 骨格追加）、または**スキルどおりのチェックリスト確認**を行う。

## チャットでの返答

- 変更した節の要旨（**どの節を無料に残し、どの節を筆者メモへ動かしたか**）
- 保存／更新したファイルのリポジトリ相対パス
- 残っている「結論の人質」や「観察／判断の混線」があれば**一言だけ**指摘

# note 三層パック（本編＋補助＋筆者メモ・有料想定）

**本編**（無料で論点まで完結）＋**補助**（無料で用語・読み順・レイヤー）＋**筆者メモ**（note 上で有料ブロック想定）の役割分担で `note/` 底稿を整理する。実行時は **`.cursor/skills/note-trio-pack/SKILL.md`** に従う。

## いつ使うか

- 「難しい」対策で**補助記事**を増やしたいとき
- 既存本編に**読み順**と**補助へのリンク**、**主な参照（出典）の直後**に**筆者メモ（有料想定）**の見出しを足したいとき
- 新規トピックで、最初から三層でファイル名を揃えたいとき

## 入力

- **本編**のパス（必須）。任意で**補助**のファイル名・**筆者メモ**に入れたい論点リスト。

## 出力

- ユーザーの指示に従い、**補助 `.md` の新規作成**、**本編の追記**、または**スキルどおりのチェックリスト確認**を行う。

## 関連

- `.cursor/commands/note-article-from-brief.md`（本編単体）
- `.cursor/commands/overseas-it-news-deep-dive.md`（ブリーフ）

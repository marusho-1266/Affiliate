## 複数トピックをまとめる場合

1. トピックごとに **H2 で分割**し、各ブロックに背景／目的／効果の小見出しを付ける。  
2. 冒頭に **1ページ相当のエグゼクティブサマリ**（全体3〜5行）を置く。

## 効果の書き方の例（ITセキュリティ）

- **短期**: CISA バインディングオペレーショナルディレクティブ（BOD）対象組織でのパッチ適用率、侵害報道の増減  
- **中期**: 製品ロードマップ（SAML 周りの設計変更）、運用チェックリストの業界標準化  
- **長期**: ゼロトラスト・ID基盤投資、クラウド境界アプライアンス市場

## 効果の書き方の例（資金調達・AI）

- **短期**: メディア露出、採用・パートナー交渉、競合のカウンター発表  
- **中期**: API 価格・利用規約、スタートアップの追随調達、GPU 供給配分  
- **長期**: 規制（著作権・エネルギー）、オープンモデルとクローズドの勢力図

# 公開前チェックリスト

note 記事を公開する前に確認するチェックリストです。

## 本文

- [ ] 無料側だけで、何が起きたか／何が分からないかが伝わる
- [ ] 無料の締めが「詳しくは有料で」だけになっていない
- [ ] 有料ブロックには、筆者の判断・比較軸・反証・打ち手・観測点がある
- [ ] 報道・一次事実そのものを有料側に隠していない
- [ ] 出典・参照リンクが整理されている

## 有料ブロック

- [ ] 冒頭に、この章で筆者が引き受ける判断を書いた
- [ ] 取材の有無を書いた
- [ ] 投資・法務・医療など、扱わない判断領域を明記した
- [ ] シナリオ分岐は「確度」ではなく「注視順」として書いた
- [ ] セルフ反証がある
- [ ] 次に見る指標が読者にも確認できる粒度になっている

## 派生

- [ ] サムネ文言を作った
- [ ] 公開当日のSNS 1枚投稿を作った
- [ ] 翌日以降のSNS連投を作った
- [ ] ラジオ／ポッドキャスト台本の要否を判断した
- [ ] 候補ログに続報・関連テーマを1行追加した

## note 公開設定

- [ ] タイトルとサブタイトルが購入後の成果物を示している
- [ ] カバー画像を設定した
- [ ] タグを3〜5個設定した
- [ ] マガジンに紐づけた
- [ ] 有料ブロック範囲を確認した
- [ ] 公開作業メモを本文から削除した

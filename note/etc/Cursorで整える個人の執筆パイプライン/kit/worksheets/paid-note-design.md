## 4. 無料から有料への地図の1文

無料末尾に置く文:

> ここから先の筆者メモでは、上の事実整理を前提に、筆者がどのシナリオを先に注視するか、どの評価軸で比較するか、読みが崩れる条件は何かを整理します。

## 5. 免責

- 当事者への取材の有無:
- 扱わない判断領域（投資／法務／医療など）:
- 読者に自分の文脈で読み替えてほしい点:

## 6. 有料化してよいかの最終判定

- [ ] 無料側だけで、何が起きたかが分かる
- [ ] 無料側だけで、誤読しやすい点が分かる
- [ ] 有料側に、筆者の重みづけがある
- [ ] 有料側に、セルフ反証がある
- [ ] 有料側に、次に見る指標がある
- [ ] 「詳しくは有料で」で結論を人質にしていない

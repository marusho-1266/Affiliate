# 完成版フォルダ構成サンプル

自分の執筆用フォルダに、次のような構成を作ります。

```text
writing-workspace/
  .cursor/
    commands/
      weekly-report.md
      meeting-minutes.md
      reading-note-deep.md
      social-post.md
      social-thread.md
      radio-script.md
      thumbnail-copy.md
    skills/
      editorial-voice/
        SKILL.md
      output-quality-gate/
        SKILL.md
  reports/
    candidates/
      backlog.md
  worksheets/
    paid-note-design.md
    series-operation.md
  notes/
    drafts/
    published/
```

## 役割

- `.cursor/commands/`: スラッシュで呼び出す作業手順。
- `.cursor/skills/`: 複数の Command にまたがる共通ルール。
- `reports/candidates/backlog.md`: 書くかもしれない題材の候補ログ。
- `worksheets/`: 有料note設計や連載運用の下書き。
- `notes/drafts/`: 本稿のドラフト置き場。
- `notes/published/`: 公開済み記事の控え。

## 最小構成

最初から全部入れる必要はありません。まずは次の4つで始めてください。

- `.cursor/commands/weekly-report.md`
- `.cursor/commands/meeting-minutes.md`
- `.cursor/skills/editorial-voice/SKILL.md`
- `.cursor/skills/output-quality-gate/SKILL.md`

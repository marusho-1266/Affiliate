# 出力の最終ゲート（output-quality-gate）

## このスキルの主管範囲

このスキルは、**出力を返す直前の最終確認**を主管する。文体・禁止・帰属は `editorial-voice` スキルに譲る。

## 必ず通す3つのゲート

### ゲート1: 見出しの順序

- Command 側で指定された**見出しの順序と数**が、出力に反映されているか。
- 勝手に節を追加・削除していないか。
- 呼ばれていない「その他」「補足」を足していないか。

### ゲート2: 入力にない情報の混入

- 入力に含まれていない**固有名詞・日付・数値**を、推定で書いていないか。
- 一般常識の範囲で補った箇所があるなら、「要確認」または「一般には〜とされる」を添えているか。

### ゲート3: 曖昧さのチェック

- **主語が分からない**文がないか。
- **期限・担当・成果物**のうち、業務系の出力で欠けているものがないか。
- 読者（翌日の自分・同僚）が**1回読みで意味を取れる**文になっているか。

## 出力前の最終チェックリスト

- [ ] ゲート1: 見出しの順序と数を守っている
- [ ] ゲート2: 入力にない固有情報は「要確認」が添えてある
- [ ] ゲート3: 主語が省略された文が多すぎない
- [ ] ゲート3: 期限・担当・成果物が揃っている（該当する場合）

上記のいずれかが通らない場合は、出力を**返す前に自己訂正**して再生成する。

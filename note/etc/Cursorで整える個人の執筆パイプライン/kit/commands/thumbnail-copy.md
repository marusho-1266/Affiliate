# 本稿からサムネ文言を作る

以下の入力（本稿）をもとに、**サムネ画像に載せる文言**を案出ししてください。

## 入力

本稿の本文（Markdown）。入力の境界は Command 呼び出し直後の `---` 以降。

## 出力の型

3案を次の形式で出す。

1. **主題コピー**（10〜18字、1〜2行）
2. **副題**（任意、15字以内）
3. **タグ帯**（記事ジャンルを示す2〜4字、例: 企業経営／AI動向／仕事術）

## 書き方のルール

- **本稿のタイトルと重複しない**。サムネは「本稿を知らない人を止める」役割。
- **数字や固有名詞**を1つ以上入れる。ただし本稿にあるものに限る。
- **疑問形・煽り形**（「本当に?」「衝撃」など）は使わない。
- 横書きを前提とする。

## 書かないこと

- 具体的な色・フォント・レイアウトの指定
- 画像生成AI向けのプロンプト

## 仕上げ

3案それぞれに **「どの読者層に刺さる想定か」**を1行で付記する。

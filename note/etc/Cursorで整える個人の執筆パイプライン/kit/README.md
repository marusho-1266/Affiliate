# Cursor 執筆パイプライン作成キット

このフォルダは、連載「Cursorで整える個人の執筆パイプライン」の購入者向け作業キットです。記事を読むだけで終わらせず、自分の執筆フォルダに Commands / Skills / ワークシートを移して使うための素材をまとめています。

## 使う順番

1. `folder-structure.md` を見て、自分の執筆フォルダに置き場所を作る。
2. `commands/` の中から、まず1本だけ `.cursor/commands/` にコピーする。
3. `skills/` の2本を `.cursor/skills/` にコピーする。
4. `worksheets/paid-note-design.md` で、有料noteの無料/有料境界を設計する。
5. 記事公開後は、派生用 Command で SNS・ラジオ・サムネ文言を作る。
6. `release-checklist.md` で公開前に確認する。

## 最初に入れるおすすめ

- `commands/weekly-report.md`
- `commands/meeting-minutes.md`
- `skills/editorial-voice/SKILL.md`
- `skills/output-quality-gate/SKILL.md`

note 有料化まで見据える場合は、次も追加してください。

- `worksheets/paid-note-design.md`
- `commands/social-post.md`
- `commands/social-thread.md`
- `commands/thumbnail-copy.md`

## 注意

このキットは、完全自動で記事を完成させるためのものではありません。最後に自分の言葉で整える前提で、構成・判断・派生の迷いを減らすための道具です。

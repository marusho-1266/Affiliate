## 1. 用語ミニ辞典

| 語 | ひとこと |
|----|-----------|
| **Codex** | OpenAI の開発者向けプロダクト。公式は「週あたり **300万人超**の開発者が利用」と記載（英語原文: *more than 3 million developers … every week*）。 |
| **computer use（コンピュータ利用）** | エージェントが**画面を見てクリック・入力する**能力の総称として本ブログで説明。**macOS で先行**、EU・UK は「soon」との記載。 |
| **SDLC** | Software Development Lifecycle（ソフトウェア開発ライフサイクル）。設計・実装・テスト・レビュー・運用にまたがる一連の工程。公式は Codex をこの全体を跨ぐパートナーとして位置づけている。 |
| **MCP** | Model Context Protocol。公式はプラグインが **skills・app 連携・MCP サーバー**を束ねる、と説明。 |
| **devbox** | リモートの開発用マシン／環境。本件では **SSH 接続**が **alpha** と記載。 |
| **gpt-image-1.5** | 画像生成・反復に使うモデル名。開発者向けドキュメントへのリンクが公式記事内にある。 |
| **rolling out（段階的ロールアウト）** | 全ユーザーに一斉ではなく、順次提供が始まること。公式は **Codex デスクトップ＋ChatGPT サインイン**から開始と説明。 |
| **alpha / preview** | 試験段階の機能。**SSH devbox** は alpha、**メモリ**は preview と明記。 |

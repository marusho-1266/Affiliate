## 1. 用語ミニ辞典

| 用語 | 短い説明 |
| --- | --- |
| **Anthropic** | Claude を提供する AI 企業。今回の主語。 |
| **Claude** | Anthropic の AI アシスタント／モデル群。法人・開発者向け利用も広がっている。 |
| **ラン・レート売上** | 直近の月次・四半期ペースを年換算した目安。会計上の年間売上そのものではない。 |
| **post-money 評価額** | 資金調達後の企業価値。pre-money と混ぜると比較がずれる。 |
| **プレエンプティブ・オファー** | 正式な資金調達プロセスより先に、投資家が持ち込む先回りの投資提案。 |
| **Series G** | Anthropic が 2026年2月に公式発表した大型調達ラウンドの呼び名。 |
| **TPU** | Google が設計する AI 計算向けチップ。Anthropic は Google・Broadcom との容量拡張を公式に説明している。 |
| **ギガワット（GW）** | 電力・データセンター規模を見る単位。AI インフラ競争では資金額と並んで重要になりやすい。 |

## 4. 本編に入る前のチェックリスト

- **評価額の数字**は、誰が言ったものかを確認する。
- **ラン・レート売上**は、会計上の年間売上ではなく年換算ペースとして読む。
- **公式発表**と**匿名ソース報道**を同じ強度で引用しない。
- **「IPO 前」「最後の私募ラウンド」**は、会社の公式計画ではなく、報道上の見立てとして読む。
- 社内資料にするなら、数字の横に **公式／報道／匿名ソース** のラベルを付ける。

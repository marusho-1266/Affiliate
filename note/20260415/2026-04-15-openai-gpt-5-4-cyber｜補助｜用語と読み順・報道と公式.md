## 1. 用語ミニ辞典

| 語・略語 | ざっくりした意味 |
|----------|------------------|
| **GPT-5.4-Cyber** | OpenAI の GPT-5.4 を、**サイバー防衛向け**に追加学習した派生。公式は **cyber-permissive**（制限を緩めた）と説明。 |
| **fine-tune（追加学習）** | すでにあるモデルを、**特定の用途向け**に追加で学習させること。 |
| **TAC（Trusted Access for Cyber）** | **信頼できる利用者だけ**が、防衛向けの強い能力にアクセスできるようにする枠組み。身元確認や段階（ティア）とセット。 |
| **ティア** | 利用できる範囲の**段階**。上の段階ほどできることが増える、というイメージ。 |
| **permissive** | AI が**お断りしにくい**設定。便利になる反面、社内ルール・契約・ログ設計がより重要になる、という文脈で出る。 |
| **ZDR（ゼロデータ保持）** | データを（原則）**ベンダに残さない**運用の考え方。公式は「**別会社のプラットフォーム経由**では制限が付きうる」と書いている。 |
| **サードパーティ** | **第三者**の会社やサービス。自社と OpenAI だけでなく、間に別の提供形態がある、という意味合い。 |
| **二用途（dual-use）** | **平和利用にも軍事などにも転ぶ**恐れがある技術として扱う、という整理。公式の文脈でよく出る。 |
| **反復的デプロイ** | **少しずつ**提供範囲を広げ、**繰り返し**運用しながら調整する、という配布の進め方。 |
| **Reuters（ロイター）** | 通信社。**電文**は見出しや並べ方に**編集方針**が乗りやすい。一次ではないが、取材ベースの報道として参照される。 |
| **一次（公式）** | 企業の**ブログ・プレス**など、当事者が直接出した文書。日付はページ表記に従う。 |
| **Mythos / Glasswing** | Anthropic の防衛文脈の製品・イニシアティブ。本編では **Reuters が並べる話のつなぎ**と、**OpenAI 公式（Anthropic 名なし）**を切り分けるのがポイント。 |
| **Codex Security** | OpenAI が公式で触れる、コードまわりのセキュリティ支援の取り組み。本編の数値主張の出典は公式投稿内。 |
| **PoC** | **試験導入**（Proof of Concept）。小さく試してから広げる、という意味で本編でも使用。 |
| **バイナリ逆コンパイル** | 実行ファイル（バイナリ）を**解析**する作業の一種。防御の文脈で公式が例として挙げることがある。 |

## 3. 「報道」と「公式」のレイヤー

同じ日でも、**情報の出どころ**が違います。

| レイヤ | 例（本件） | 読み手が注意すること |
|--------|------------|------------------------|
| **通信社の報道** | Reuters の見出し・リード | **競合のタイミング**など、読みやすい並べ方が入りうる。**因果の断定はしない**。 |
| **当事者の公式** | OpenAI のブログ（4/14） | **自社のプログラム**の説明が中心。**他社名が出ない**こともありうる。 |

**混同しやすいポイント:** 「Mythos の一週間後」は **Reuters 側の文脈**であり、OpenAI 公式が Anthropic と**同じ構図で書いているとは限らない**、と本編でも述べています。

## 0. Cerebras Systems とは — 会社の輪郭（公開情報ベース）

### 一言

**Cerebras Systems** は、**AI 向けに「シリコンウエハーに近い面積」を一枚の巨大プロセッサ（ウエハースケール）として使う** **Wafer Scale Engine（WSE）** と、それを載せた **CS シリーズ**（例: **CS-1 → CS-2 → CS-3**）を開発し、**学習・推論**のワークロード向けに**オンプレの大規模システム**や**クラウド上の従量課金**などを提供する、**米国の AI コンピューティング企業**です。メディアでは **Nvidia の GPU 系**と同じ「AI インフラ市場」で語られることが多い一方、**チップの切り方（多数の小さな GPU ダイ vs ウエハー一枚）**は設計思想が異なります。

### 設立・拠点（公式・百科の整理）

- **2015年設立**。創業者は **Andrew Feldman, Gary Lauterbach, Michael James, Sean Lie, Jean-Philippe Fricker** とされる（[Cerebras — Company](https://www.cerebras.ai/company)）。創業チームは **SeaMicro** などで共に働いた経緯が語られることがあります（二次: [Wikipedia — Cerebras](https://en.wikipedia.org/wiki/Cerebras)）。
- **本社**は**カリフォルニア州サニーベール**（SEC 提出書類・会社サイト記載）。百科では **サンディエゴ・トロント・インド（バンガロール）** などの拠点が列挙されています（**最新は会社開示・採用ページで要確認**）。

### 製品・提供形態（公式の言い方）

[Cerebras — Company](https://www.cerebras.ai/company) では、研究・暗号・エネルギー・エージェント系 AI などの分野で **CS-2 / CS-3** を使う例に触れ、**従量課金のクラウド**で開発者・企業がアクセスできる旨が述べられています。世代のマイルストーンとして、**2019年 WSE-1 と CS-1**、**2021年 WSE-2 と CS-2**、**G42 との Condor Galaxy ネットワーク（2023年頃の記述）**、**2024年 WSE-3 と CS-3** などが年表に載ります（**性能・順位づけの文言は会社の主張**であり、**自社ワークロードでの検証**が別問題です）。

### 資金調達・上場の文脈（「話題の軸」だけ）

累計調達額や評価額、**2024年の IPO 届出・2025年の取り下げ・G42 と CFIUS・2026年の再提出**などは、**報道と百科に整理が出る**題材です。**契約条件・売上構成・リスクの正本**は **Form S-1** と会社の法定開示にあります。

### 読み分けのメモ

| 観点 | メモ |
|------|------|
| **チップ企業だけか** | **WSE＋システム（CS）＋クラウド／大規模クラスタ**まで含めて語られることが多い。 |
| **差別化の軸** | **ウエハースケール**によるオンチップのメモリ・帯域の取り方。**「HBM に依存しない」**等の言い回しは**製品ストーリー**と**実測ベンチ**を分けて読む。 |
| **ニュースの中心になりやすい相手先** | **G42（UAE）**、**OpenAI** など。**地政学・顧客集中**は **S-1 のリスク要因**で一次確認。 |

## 1. 用語ミニ辞典

| 語 | 平易な意味 |
|----|------------|
| **IPO** | 未公開株を公開市場に出して資金を集める一連の手続き。 |
| **Form S-1** | 米国 SEC に提出する**有価証券届出書**の代表的様式。事業説明・リスク要因・財務表が載る。修正案が出ることもある。 |
| **ウエハースケール（wafer-scale）** | シリコン**ウエハーに近い面積**を、**一枚のプロセッサ**として使う設計思想。従来の「ウエハーから多数の小チップを切り出す」とは対照的に語られる。 |
| **WSE（Wafer Scale Engine）** | Cerebras の**ウエハースケール AI プロセッサ**の製品名。世代として **WSE-1 / 2 / 3** が公式年表に登場。 |
| **CS（例: CS-1, CS-2, CS-3）** | WSE を搭載した**システム／製品ライン**（公式の呼び方に従う）。 |
| **推論（inference）** | 学習済みモデルが**問い合わせに応じて出力する**フェーズ。学習（training）と対で使う。 |
| **学習（training）** | モデルをデータで更新するフェーズ。 |
| **HBM** | 高帯域幅メモリ。GPU 周辺で帯域を稼ぐ部品で、**供給・コスト**が市場の話題になりやすい。 |
| **Nvidia / GPU** | AI インフラでは**事実上の基準**として語られることが多い。**ベンチ条件・ワークロード**が違うと比較はすれ違う。 |
| **CFIUS** | 米国が**外国投資を国家安全保障の観点で審査**する枠組み。**個別案件のクリア**と**将来の再審査**は別物として読む。 |
| **G42** | UAE 系のテック／AI 企業群として報道・百科に登場。**Cerebras の大型インフラ連携**（Condor Galaxy 等）の文脈で語られる。 |
| **Nasdaq** | 米国の証券取引所の一つ。ティッカー（銘柄コード）を**目論見書・プレス**で確認。 |
| **主幹事（アンダーライター）** | IPO で売出しを主導する投資銀行群。**Reuters やプレス**に列挙されることがある。 |
| **RPO（Remaining Performance Obligations 等）** | 契約に基づき**将来に認識される収益**のような指標が、開示で語られることがある（**定義は S-1 の注記**）。 |
| **関連当事取引** | 支配株主・役員・関連会社との取引。**目論見書の注記**で確認するのが安全。 |
